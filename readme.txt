This plugin allows you to specify redirections from one URL to another.

== Description ==

You can customize how many redirections you want to enter. Simply specify the number of redirects you want to create, and the plugin will generate a form where you can enter the source and destination URLs for each redirection.

With this plugin, you can easily manage redirections on your WordPress site and ensure that your visitors are always directed to the correct pages. Whether you're moving content to a new URL, fixing broken links, or simply want to redirect visitors from an old URL to a new one, this plugin makes it easy to set up redirections in just a few simple steps.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simpler_redirects` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the "Redirects" menu point to configure the plugin

== Frequently Asked Questions ==

= How do I set up a redirection? =
To set up a redirection, use the "Redirects" menu point in your WordPress backend. Specify the number of redirects you want to create, and the plugin will generate a form where you can enter the source and destination URLs for each redirection.

= Can I redirect to an external URL? =
Yes, you can redirect to an external URL by entering the full URL in the destination field of the redirection form.

= Can I redirect multiple URLs to the same destination? =
Yes, you can use the same destination URL for multiple redirections. Simply enter the same URL in the destination field for each redirection you want to create.
